Simulation Methods

The methods of the Simulation class are listed below. For a complete list of Simulation class members, see the Simulation Members topic.

Public Static (Shared) Methods

Interrupt Interrupt wird verwendet um eine Methode eines bestimmten Objektes zu unterbrechen. Es wird die erste gefundene Methode unterbrochen!
InterruptAll Hier werden alle aktiven Methoden eines bestimmten Objektes unterbrochen.
SimTime Gibt aktuelle Simulationszeit zurück.
StartSimulation Starten der Simulation
StopSimulation Stoppen der Simulation
Tell Tell-Methode Hier wird ein neues Objekt vom Typ SimObj angelegt, dem dann die angegebenen Paramater entsprechend zugewiesen werden. Anschließend wird dieses Objekt zur Verarbeitung in die Schedulerliste aufgenommen. Die Umsetzung der Tell-Methode erfolgt über ein Delegate, damit der Scheduler, eben stellvertretend diese Methode zu gegebener Zeit ausführen kann.
Wait Wait dient zum Warten einer bestimmten Zeit t. Die aufrufende Methode suspendiert also ihre Arbeit für einen gewissen Zeitraum.
WaitForOverloaded. WaitFor bedeutet, das das aktuelle Objekt seine Arbeit suspendiert und solange wartet bis die aufgerufene Methode tm (in diesem Fall eine Tell-Methode) beendet ist.
