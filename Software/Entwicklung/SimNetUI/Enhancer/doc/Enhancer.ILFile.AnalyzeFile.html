ILFile.AnalyzeFile Method 

Hier wird IL-Datei analysiert. Für jede Zeile wird ein ILLineElement angelegt und im Array lines abgespeichert. Es wird nach bestimmten Direktiven gesucht, in diesem Fall speziell nach dem Beginn der Methoden (.method). Für jede Methode wird ein ILMethodsElement angelegt und im Array methods abgespeichert. Über die Methode ReadToEnd() wird die Methode analysiert.
