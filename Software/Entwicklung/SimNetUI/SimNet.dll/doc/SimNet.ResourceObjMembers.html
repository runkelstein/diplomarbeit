Public Instance Methods

CreateResource Initialisiert das Resource-Objekt mit der angegeben Anzahl von Resourcen. Es wird also die Kapazität der Resource festgelegt.
Equals (inherited from Object) Bestimmt, ob das angegebene Object und das aktuelle Object gleich sind.

GetAeq Gibt Anzahl der Eintritte in Warteschlange zurück.
GetAeq0 Gibt Anzahl der Eintritte in Warteschlange zurück, bei denen Wartezeit gleich null ist.
GetHashCode (inherited from Object) Fungiert als Hash-Funktion für einen bestimmten Typ, der sich für die Verwendung in Hashalgorithmen und Hashdatenstrukturen, z. B. einer Hashtabelle, eignet.
GetMa Gibt mittlere Auslastung der Ressource zurück.
GetMaxi Gibt maximalen Inhalt der Ressource zurück.
GetMaxq Gibt die maximale Länge der Warteschlange zurück.
GetMb Gibt mittlere Belegungsdauer zurück.
GetMi Gibt mittleren Inhalt der Ressource zurück.
GetMlq Gibt die mittlere Länge der Warteschlange zurück.
GetMv Gibt mittlere Verweilzeit der Ressource zurück.
GetMvq Gibt mittlere Verweilzeit der Warteschlange zurück.
GetMvq0 Gibt mittlere Verweilzeit der Warteschlange zurück.
GetPendingList Gibt Liste der auf Ressourcen wartenden Objekte zurück.
GetResource Ist eine Kombination aus TimedGive und PriorityGive. Die anfragende Methode wird unterbrochen wenn die Resourcen nicht innerhalb der angegebenen Zeiteinheiten zur Verfügung stehen. Die Anfragen werden hier aber auch nach Priorität in die Warteliste eingeordnet.
GetResourceList Gibt Liste der Objekte, die gerade Ressourcen verwenden zurück.
GetType (inherited from Object) Ruft den Type der aktuellen Instanz ab.
GetZI Gibt das Zeitintegral der Ressourcen zurück.
GetZIQ Gibt Zeitintegral der Warteschlange zurück.
Give Stellt eine Anzahl von benötigten Resourcen dem anfragendem Objekt zur Verfügung. Die aufrufende Methode wird solange geblockt, bis die Resourcen verfügbar sind. Die Anfragen werden in der Warteliste (PendingList) nach dem Prinzip Frist-In First-Out verarbeitet.
IncrementResources Vergrösseren der Kapazität des Resource-Objektes um den angegebenen Wert.
PriorityGive Hat die selbe Aufgabe wie Give, mit dem Unterschied das die Anfrage nach der Priorität behandelt wird. D.h. je höher die Priorität, desto weiter vorn in der Warteliste wird die Anfrage eingeordnet.
TakeBack Gibt die angegebene Anzahl von Resourcen wieder frei.
TimedGive Hat die selbe Aufgabe wie Give, mit dem Unterschied das die aufrufende Methode unterbrochen wird, wenn in der angegebenen Zeit timePeriod die Resourcen nicht zur Verfügung gestellt werden können.
ToString (inherited from Object) Gibt einen String zurück, der das aktuelle Object darstellt.
